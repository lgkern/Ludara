Neste jogo, o jogador será um Dragão. Este Dragão vive em uma região rural medieval fantástica. O Dragão precisa se alimentar para sobreviver e é um Dragão carnívoro (versão vegetariana será um possível desenvolvimento futuro se houver público) que irá se alimentar de  animais disponíveis no cenário. Existem animais selvagens que surgem em qualquer lugar no mapa e animais domésticos vinculados a vilas. O jogador deve tentar se manter vivo o máximo de tempo possível para atingir uma pontuação mais elevada, para tanto deve manter o Dragão alimentado e evitar os ataques dos humanos que vivem nas vilas onde eventualmente buscará comida. O jogo acaba quando o Dragão morre.
